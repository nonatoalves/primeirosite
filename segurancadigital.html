Política de Privacidade (resumo)

Os dados fornecidos via formulários são utilizados apenas para contato e não são compartilhados com terceiros sem consentimento. Dados sensíveis não são solicitados na forma pública deste site. Para dúvidas específicas, consulte a política completa disponível no rodapé.

Boas Práticas Aplicadas

  • Comunicação criptografada via HTTPS/TLS (Let's Encrypt).
  • Versionamento do código-fonte no GitHub para rastreabilidade e auditoria.
  • Proteções front-end contra XSS e injeções (sanitização e validação de inputs).
  • Hospedagem em servidor confiável com monitoramento básico e backups periódicos.
